***Updated 7/15/2014***

-This code is used for parsing, interpretting, fitting, and exporting spectrometry data from measurements of
optical cavity resonators. 
-The data is modeled as lorentzian and is fitted as such assuming there is either 1 peak or 2 
intersecting peaks. 
-The data is exported as an excel in a very specific format useful for the lab it was written for
-More detailed descriptions are included for each file.
